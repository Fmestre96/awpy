Installation
============

To install csgo, you must have Go (the programming language) installed. You need a version greater than 1.16, and you can visit this link to install the latest Golang_.

:: _Golang: https://go.dev/dl/

Once you have Golang installed, you can check your version in command line by using the command ``go version``. After this, using ``pip``, simply run

``pip install git+https://github.com/pnxenopoulos/csgo@main``
