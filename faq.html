Frequently Asked Questions (FAQs)

This is a nonexhaustive list of frequently asked questions.

Do I need Golang to run this

You don’t need Golang to install this library or access any part of it. However, you need Golang 1.16+ to parse demofiles using this library.

The parser returns weird rounds or data

Please note that the parser parses everything in a demo. This means that you may have rounds from the warmup, rounds that may have ended in a draw, and other odd-looking rounds. Try using the DemoParser.clean_rounds() method to clean up. Note that this is not going to be 100 percent perfect. If you still have bad data, ask in the Discord.

Data I want is missing

Feel free to open an issue or to ask in the Discord, and we can see if such a feature is possible.

Where can I get documentation for the parsed data structure/keys?

Please look at this documentation.
